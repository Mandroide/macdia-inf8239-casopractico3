# Directrices para la anotación de incidente de tránsito

## Criterios de severidad

| Severidad | Criterios (ejemplos)                                                                                                                                                                                           |
|-----------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Alta      | <ul><li>Heridos/graves/muertos</li><li>Cierre total de vía</li><li>Duración > 2-3 horas</li><li>Intervención policial/bomberos</li><li>Palabras: "grave", "muerto", "hospital", "cerrado", "colapso"</li></ul> |
| Media     | <ul><li>Retención significativa (>30 min)</li><li>Carril bloqueado</li><li>Desvíos necesarios</li><li>Palabras: "lento", "congestionado", "atascado", "varios carros"</li></ul>                                |
| Baja      | <ul><li>Retención leve (<15 min)</li><li>Obstrucción menor</li><li>Sin afectación mayor</li><li>Palabras: "poquito", "pasando lento", "se mueve"</li></ul>                                                     |

## Criterios de tipo de incidente

| Tipo de incidente | Criterios (ejemplos)                                                                                   |
|-------------------|--------------------------------------------------------------------------------------------------------|
| accidente         | <ul><li>accidente</li><li>choque</li><li>colisión</li><li>volcadura</li></ul>                          |
| tapón             | <ul><li>tapón</li><li>congestion</li><li>trancón</li><li>tráfico lento</li><li>tráfico denso</li></ul> |
| bloqueo           | <ul><li>vía bloqueada</li><li>carretera cerrada</li><li>no pasa</li><li>bloqueo</li></ul>              |
| lluvia            | <ul><li>lluvia fuerte</li><li>inundación</li><li>pavimento mojado</li></ul>                            |
| obstrucción       | <ul><li>árbol caído</li><li>cable</li><li>piedra</li><li>escombro</li></ul>                            |
| manifestación     | <ul><li>piquete</li><li>protesta</li><li>marcha</li></ul>                                              |
